Descrição Geral

O projeto consiste na criação de uma arquitetura serverless na Azure para processar automaticamente arquivos de dados financeiros enviados pelos usuários. O fluxo começa quando um arquivo CSV é carregado no container RAW do Azure Blob Storage. Esse evento aciona uma Azure Function configurada com um Blob Trigger, responsável por ler o arquivo, processar os dados e gerar insights financeiros, como análises de gastos, alertas e recomendações de economia.
